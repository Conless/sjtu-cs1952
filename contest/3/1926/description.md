# 看不懂的代码

你点了「发射」。什么都没有发生。看来他们的代码写错了。

通过其他手段，你把控制系统的源代码全部拉了下来。代码写得非常乱，你完全无法理解它们都对应着什么功能。你只知道调用某些函数可以触发发射。

### 提示

输入的格式是这样的: 首先输入一个类名，然后依次输入这个类的所有成员变量。若成员变量希望留空，可以直接填 null。例如这样的类结构:

```cpp
struct A {};
struct B {
  A *a;
};
struct C {
  B *b;
  A *a;
};
```

输入

```
C
B null
null
```

相当于

```cpp
new C { new B { nullptr }, nullptr };
```

本题满分为 200 分。
